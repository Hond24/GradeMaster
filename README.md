# GradeMaster
学生管理システム

## 概要

このプログラムは、コンソールベースの学生管理システムです。学生の名前と点数を管理し、追加・削除・更新・平均点の計算ができます。

---

## 使い方
1. プログラムを実行する
2. メニューから操作を選択する

 各メニューの詳細

1. 学生の追加: 名前と点数を入力して登録

2. 学生の削除: 学生の名前を入力して削除

3. 点数の更新: 学生の名前を入力し、新しい点数を設定

4. 平均点の計算: 登録されている学生の平均点を計算

5. 学生一覧の表示: 全ての登録学生の名前と点数を表示

6. 終了: プログラムを終了
